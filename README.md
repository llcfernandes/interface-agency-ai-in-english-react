# 🚀 agency.ai

Modern, animated and fully responsive digital agency landing page built with React 19, Tailwind CSS v4 and Framer Motion.

Designed to simulate a real-world SaaS or digital agency website, the project focuses on polished user experience, interactive UI elements, scroll-based animations and modern frontend development practices.

Key goals of the project:

* Build a realistic agency landing page
* Create an engaging user experience through animations
* Implement reusable React component architecture
* Explore advanced Tailwind CSS capabilities
* Integrate external form services

## 🎯 Technical Highlights

### Custom Animated Cursor

A custom two-layer cursor replaces the browser's default cursor.

Features:

* Precise cursor tracking
* Smooth interpolation (LERP)
* requestAnimationFrame rendering loop
* Performance-oriented implementation

---

### Interactive Spotlight Effect

Each service card dynamically tracks mouse movement and renders a radial gradient that follows the cursor position.

Implementation includes:

* Mouse position tracking
* Dynamic gradient generation
* Real-time visual feedback
* GPU-friendly rendering

---

### Theme System

The theme layer provides:

* Dark mode
* Light mode
* Preference persistence
* System theme detection

Theme state is synchronized with localStorage and automatically applied to the HTML root element.

---

### Animation Strategy

Animations are orchestrated using Framer Motion.

Implemented techniques:

* whileInView triggers
* staggerChildren sequencing
* viewport-based rendering
* one-time animation execution
* smooth section transitions
